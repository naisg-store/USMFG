# Social Media Font Generator
**A Mayteck C.S. Utility**
*Creative Sites & Smarter Solutions*

Welcome to the Social Media Font Generator. We built this tool to solve a simple, daily frustration: standard text formatting gets aggressively stripped away by social media platforms. 

This utility uses universally recognized Unicode symbols to generate aesthetic typography—like Gothic, Cursive, and Glitch text—that survives the filters on Instagram, TikTok, and Twitter. 

**Why we built it:**
* **Zero Friction:** No bloated code, no popups, no waiting. Just instant translation.
* **Privacy First:** The translation engine runs entirely natively in your browser. We never see, store, or track what you type.
* **Maximum Impact:** Elevate digital profiles with typography that actually stands out.
